# 撰写文章

## 快速入门

使用 `drafts.md` 中的信息(如果有需要，可以去 `drafts` 目录查看原始信息)，面向科技爱好者，撰写一份引人入胜的文章。

## 写作导向

- 关注独特的观点和深入的分析和热门的工具。
- 摈弃营销类语言，确保内容真实可信。

## 要求

- 文章使用 Markdown 语法，包含 Front matter 头部 。
- Front matter 包含：`title`、`date`、`summary`、`tags` 字段。
- 文章有开场白和结束语，`summary` 中包含一份简短的文章介绍。
- 文章包含 `资讯`, `模型` 和 `工具` 三个部分。
- 文章的信息必须来自 `drafts.md`，不得包含其他来源的信息。
- 保留文章的原始链接。

## 写作风格

- 语言简洁明了，适合广泛读者。
- 语气专业但不失亲切，易于理解。
- 可以包含 Emoji，但是要适量使用，确保不影响阅读体验。
